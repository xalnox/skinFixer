# FixMySkin

## A legacy fabric 1.4.7 mod that provide multiple utilities around skins

### Current feature

##### The following feature requires an internet connection

- Fix the skin not loading (singleplayer/multiplayer)
- Fix the cape not loading (singleplayer/multiplayer)
- New 1.8+ skin system
- Slim skin support (partial)
